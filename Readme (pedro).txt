Sistema de Calculadora:

Este é um sistema de calculadora remota onde o cliente tem a possibilidade de fazer operações matemáticas juntamente com valores numéricos para o servidor. 
O servidor realizará a operação solicitada e retornará o resultado para o cliente.
Execute o arquivo server.py na máquina do servidor e o client.py na máquina do cliente.
Insira os números para a operação escolhida.
O cliente enviará a operação e os números para o servidor e processa a operação e envia o resultado de volta para o cliente.
O cliente exibe o resultado recebido do servidor.
